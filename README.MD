```

    ts timestamp without time zone,
    tsz timestamp with time zone
);

```

```

```

```

ğŸ” Date Casting
Return only date:
```

```

```

-- Output: 13:12:19.780626

Return timestamp:
```

-- Output: 2025-05-19 13:12:19.780626
```
